## tip
把代码拉下来安装完依赖即可运行，建议使用的`node`版本为`16`以上，不依赖于任何需要从服务器读取配置，配置已内置于项目中，如果对你有所帮助的话，希望可以点一个`star`。逆水行舟，不进则退。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
